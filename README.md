# Evolutivo Ventas Usados — Enero a Junio

Reporte con 3 vistas:
- **Detalle por mes**: sucursales x indicadores, filtrable por mes, resalta cumplimiento de meta.
- **Evolución por sucursal**: todos los indicadores por sucursal a lo largo de Enero-Junio, incluye composición de venta (total/digital/diferencia) y Vendedores Disponibles.
- **Tendencia (gráficos)**: gráficos de línea filtrables por sucursal.

Fuente: `Ventas Usados Enero a Junio..xlsx` + `vendedores.xlsx` (Vendedores Disponibles, excluyendo cuenta de sistema y jefes de sucursal).

Ver online: https://barbaracolleg-art.github.io/evolutivo-ventas-usados-enero-junio/
